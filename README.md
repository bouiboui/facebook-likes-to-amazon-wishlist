### Required
This project requires you to install Zero Server as a global package
```sh
yarn global add zero
```
### Install
```sh
# (Recommended) Read the blog post first (see Github repository URL)
# Modify the .env file with the correct values
yarn
```
### Start
```sh
yarn start
```
